# Store Audits

## Назначение

Документ описывает проверки магазинов.

## Основные требования

Система должна поддерживать создание проверки, выбор магазина, указание проверяющего, ответы по пунктам проверки, оценку магазина, оценку сотрудников, комментарии и историю проверок.

## Правила

- Проверка сохраняет дату, проверяющего, ответы и оценку.
- Проверка не изменяет смену напрямую.
- Результат проверки может создавать задачи или влиять на аналитику, если это будет утверждено отдельным правилом.
