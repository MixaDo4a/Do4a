## Чек-лист как основа окладной части

Проверяющий проводит чек-лист и выявляет нарушения. Каждый пункт чек-листа оценивается по 10-балльной шкале:

- 10 - все идеально;
- 9 - незначительные упущения;
- 1 - все очень плохо.

У каждого пункта чек-листа есть вес в рублях. Для каждого пункта рассчитывается денежный результат:

```text
результат пункта = вес пункта / 10 * оценка
```

Все результаты пунктов складываются. Полученная сумма является окладной частью за одну рабочую смену по конкретному чек-листу.

За месяц одному сотруднику может быть проведено несколько чек-листов. Для зарплаты используется средний результат всех чек-листов сотрудника за расчетный месяц.

```text
окладная часть за 1 смену = среднее значение сумм всех чек-листов сотрудника за месяц
оклад за месяц = количество смен * окладная часть за 1 смену
```

## Требует уточнения

- Кто может изменять веса пунктов чек-листа.
- С какого момента новая версия весов начинает действовать.

## Статусы сотрудников в чек-листе

В чек-листе используются разные веса пунктов для разных статусов сотрудников:

- Падаван - менеджер на испытательном сроке;
- Бывалый - менеджер, прошедший испытательный срок.

Один и тот же пункт чек-листа может иметь разный вес в рублях для Падавана и Бывалого. При расчете окладной части используется вес, соответствующий статусу сотрудника на момент проведения чек-листа.

## Отсутствие чек-листов за месяц

Если за расчетный месяц сотруднику не проведено ни одного чек-листа, для расчета окладной части используется максимальный результат чек-листа для его текущего статуса.

Это означает, что отсутствие проверки не должно ухудшать зарплату сотрудника.
