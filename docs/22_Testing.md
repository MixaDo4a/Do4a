# Testing

## Назначение

Документ описывает подход к тестированию.

## Обязательные зоны тестирования

- Расчет KPI.
- Расчет зарплаты.
- Жизненный цикл смены.
- Права доступа.
- RLS-политики.
- Импорт OFD.
- Cron-задачи.
- Snapshots.

## Принципы

- Финансовая логика требует тестов.
- Роли и доступ требуют тестов.
- Миграции должны быть проверяемыми.
- Для важных правил нужны тест-кейсы со ссылкой на BR-ID.
