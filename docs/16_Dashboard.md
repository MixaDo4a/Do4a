# Dashboard

## Назначение

Документ описывает аналитические экраны и панели.

## Черновые области аналитики

- Продажи.
- KPI.
- Смены.
- Задачи.
- Проверки.
- Зарплата.
- Активность сотрудников.

## Требует уточнения

- Набор метрик для каждой роли.
- Периоды анализа.
- Фильтры: магазин, город, сотрудник, период.
- Ограничения доступа к аналитике.
