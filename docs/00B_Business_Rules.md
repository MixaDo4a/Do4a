### BR-081. Проверяющий оценивает пункты чек-листа по 10-балльной шкале

Описание: Каждый пункт чек-листа оценивается проверяющим по шкале от 1 до 10, где 10 означает идеальное состояние, 9 - незначительные упущения, 1 - очень плохое состояние.

Причина: Оценка чек-листа влияет на окладную часть зарплаты.

Последствия нарушения: Окладная часть будет рассчитана по неподтвержденной системе оценок.

Связанные правила: BR-032, BR-067, BR-082.

Статус: Confirmed.

### BR-082. Каждый пункт чек-листа имеет вес в рублях

Описание: У каждого пункта чек-листа есть денежный вес, который участвует в расчете окладной части.

Причина: Разные нарушения имеют разную значимость для зарплаты.

Последствия нарушения: Все пункты будут влиять на оклад одинаково, что противоречит текущей логике.

Связанные правила: BR-067, BR-081, BR-083.

Статус: Confirmed.

### BR-083. Результат пункта чек-листа считается по формуле вес / 10 * оценка

Описание: Денежный результат пункта чек-листа рассчитывается как вес пункта, деленный на 10 и умноженный на оценку проверяющего.

Причина: Эта формула переводит 10-балльную оценку в денежную часть оклада.

Последствия нарушения: Окладная часть будет отличаться от текущей таблицы.

Связанные правила: BR-066, BR-067, BR-081, BR-082.

Статус: Confirmed.

### BR-084. Окладная часть за чек-лист равна сумме результатов пунктов

Описание: После расчета всех пунктов чек-листа их денежные результаты складываются и формируют окладную часть за одну рабочую смену по этому чек-листу.

Причина: Текущая таблица рассчитывает окладную часть как сумму всех пунктов.

Последствия нарушения: Расчет оклада за смену будет неверным.

Связанные правила: BR-083, BR-085.

Статус: Confirmed.

### BR-085. Для зарплаты используется средний результат чек-листов за месяц

Описание: Если сотруднику за месяц проведено несколько чек-листов, для расчета зарплаты используется среднее значение результатов всех его чек-листов за расчетный месяц.

Причина: Один сотрудник может иметь несколько проверок за месяц, и зарплата должна учитывать среднее качество работы за период.

Последствия нарушения: Одна проверка может неправильно определить оклад за весь месяц.

Связанные правила: BR-066, BR-067, BR-084.

Статус: Confirmed.

### BR-086. Оклад за месяц равен количеству смен, умноженному на средний результат чек-листов

Описание: Месячный оклад сотрудника рассчитывается как количество смен, умноженное на среднюю окладную часть за одну смену, рассчитанную по чек-листам месяца.

Причина: Окладная часть зависит одновременно от количества смен и качества по чек-листам.

Последствия нарушения: Окладная часть зарплаты будет рассчитана неверно.

Связанные правила: BR-066, BR-073, BR-085.

Статус: Confirmed.

Последствия нарушения: Окладная часть сотрудника после испытательного срока может быть рассчитана по неверным весам.

Причина: Компания по-разному оценивает чек-лист для сотрудников на испытательном сроке и после него.

Последствия нарушения: Окладная часть будет рассчитана без учета статуса сотрудника.

Причина: Отсутствие проверки не должно ухудшать зарплату сотрудника.

Причина: Распределение должно соответствовать фактическому участию сотрудника в работе магазина.

Последствия нарушения: Удержания будут распределены несправедливо.

Связанные правила: BR-078, BR-133.

### BR-135. Инвентаризация сотрудника считается отдельно по каждому магазину

Описание: Если сотрудник работал в нескольких магазинах, удержание по инвентаризации рассчитывается отдельно для каждого магазина.
