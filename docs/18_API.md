## Импорт данных кладовщика

Система должна поддерживать импорт из Google Таблицы, в которой работает кладовщик.

Импортируемые данные:

- просрочка;
- товары под зарплату.

После импорта данные сохраняются в PostgreSQL и используются для payroll. Google Таблица остается внешним источником ввода, но не источником истины для расчета.
