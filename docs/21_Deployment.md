# Deployment

## Назначение

Документ описывает развертывание.

## Целевая инфраструктура

- Frontend: Vercel.
- Backend: Supabase.
- Database: Supabase PostgreSQL.
- Storage: Supabase Storage.
- Edge Functions: Supabase.

## Окружения

Минимально нужны local, staging и production.

## Требует уточнения

- Домены.
- Supabase project naming.
- Переменные окружения.
- Процесс миграций.
- Процесс релиза.
